# Исследование алгоритмов и жадных методов в Python

## Описание работы

Данная работа посвящена изучению алгоритмических методов, включая жадные и рекурсивные алгоритмы, и их применению при решении практических задач программирования.

Жадные алгоритмы позволяют находить оптимальные или приближённые решения при выборе элементов с учётом ограничений. Рекурсивные алгоритмы применяются для систематического обхода структур данных и поиска всех возможных вариантов решения.

В ходе выполнения работы были рассмотрены несколько примеров применения этих методов, включая нахождение кратчайшего пути между городами, решение задачи о рюкзаке и рекурсивный поиск всех путей в графе.

---

# Цель работы

Изучить принципы работы жадных и рекурсивных алгоритмов и реализовать несколько практических примеров на языке программирования Python.

---

# Задачи работы

Для достижения поставленной цели были выполнены следующие задачи:

- изучить принципы работы жадных алгоритмов и их применение для оптимизации выбора элементов;
- реализовать алгоритм нахождения кратчайшего пути между городами;
- изучить жадный алгоритм решения задачи о рюкзаке и реализовать его;
- изучить рекурсивный подход к поиску путей в графах;
- реализовать рекурсивный поиск всех возможных путей из одного города в другой с учётом заданного условия прохождения через определённый узел;
- сравнить эффективность жадных и рекурсивных алгоритмов при решении практических задач.

---

# Используемые технологии

В процессе выполнения работы использовались следующие технологии:

- язык программирования **Python**

# Кратчайший путь между городами

Данная задача демонстрирует применение жадного алгоритма для нахождения пути, проходящего через все города с минимальной суммарной длиной. Алгоритм последовательно выбирает ближайший ещё не посещённый город и формирует путь до тех пор, пока все города не будут пройдены.

---

# Задача о рюкзаке

Задача о рюкзаке использует жадный подход для оптимального выбора предметов на основе их ценности и веса. Алгоритм позволяет определить, какие предметы следует положить в рюкзак, чтобы суммарная ценность была максимальной, не превышая при этом ограничение по весу.

---

# Рекурсивный поиск всех путей в графе

Реализован рекурсивный алгоритм для поиска всех возможных маршрутов из одного города в другой, проходящих через заданный узел. Рекурсия позволяет систематически обходить граф, избегая циклов, и выводить все допустимые пути, соответствующие заданному условию.

---

# Вывод

В ходе выполнения работы были изучены и применены различные алгоритмические подходы:

- Жадные алгоритмы позволяют находить оптимальные или приближённые решения при наличии ограничений и упрощают принятие решений в задачах выбора;
- Рекурсивные алгоритмы дают возможность систематически исследовать все возможные варианты решения в структурированных данных, таких как графы.

Реализованные задачи демонстрируют практическое применение этих методов и помогают лучше понять их принципы работы, возможности и ограничения.
